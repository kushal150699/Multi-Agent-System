## How to Run the Code in WSL

Assuming that you have already installed the required packages. 
To run the code in WSL (Windows Subsystem for Linux), follow these steps:

**Build the package**:
   ```bash
  colcon build

  source ~/.bashrc

  ros2 run campus_agents test_node
```
